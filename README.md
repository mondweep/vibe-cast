## About

An interactive web showcase for the Nagaon University AI Summit, demonstrating key agentic AI skills and techniques through hands-on browser-based demos.

## Features

- **Summit Agenda** — Full-day program covering agentic AI from fundamentals to advanced techniques
- **ReAct Agent Simulator** — Interactive demo showing the Reasoning + Acting loop
- **Multi-Agent Orchestration** — Watch Planner, Coder, Reviewer, and Tester agents collaborate
- **RAG Pipeline Visualizer** — See Retrieval-Augmented Generation in action step-by-step
- **Technique Gallery** — Code examples for ReAct, Task Decomposition, Tool Use, Memory, Multi-Agent, and Safety patterns
- **Animated Particle Background** — Interactive network visualization with mouse tracking

## Tech Stack

- Vanilla HTML5, CSS3, JavaScript (no build tools required)
- CSS Grid & Flexbox responsive layout
- Canvas API for particle animations
- IntersectionObserver for scroll animations

## Running Locally

Simply open `index.html` in a browser, or serve with any static file server:

```bash
npx serve .
```
