# Vehicle Repair Network

A platform connecting vehicle owners with repair shops and mechanics.

## About

This branch hosts the `vehicle-repair-network` project as an orphan branch within the `vibe-cast` repository. It has no shared history with the main branch and is maintained as an independent project.
