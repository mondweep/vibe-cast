# MOVE Connectivity Commander

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/mondweep/vibe-cast)

A high-impact, visual demonstration of the Tata Communications MOVE™ platform. This application acts as a "Mission Control" for global IoT connectivity, visualizing API capabilities in a premium interface.

## Features
- **Global Pulse**: Real-time 3D visualization of active connections.
- **eSIM Factory**: Visual lifecycle management for eSIM profiling.
- **Fleet Health**: Smart monitoring and alerting system.

## Getting Started

### Prerequisites
- Node.js 18+
- npm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/mondweep/vibe-cast.git
   cd vibe-cast
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000).

## Tech Stack
- **Framework**: Next.js 14
- **Styling**: TailwindCSS
- **Visuals**: Three.js / React Three Fiber
- **Icons**: Lucide React

## Deployment

### Netlify
This project is configured for deployment on Netlify.
1. Click the "Deploy to Netlify" button above.
2. Connect your GitHub account.
3. Netlify will automatically detect the `netlify.toml` and build configuration.
