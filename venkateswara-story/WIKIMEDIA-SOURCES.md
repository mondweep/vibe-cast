## How to Use These Images

### For Direct Use (CC-BY-SA)
Add attribution like:
```html
<img src="image.jpg" alt="Description">
<p class="attribution">
    Image: [Title] by [Author],
    <a href="[wikimedia-link]">Wikimedia Commons</a>,
    <a href="https://creativecommons.org/licenses/by-sa/4.0/">CC BY-SA 4.0</a>
</p>
```

### For Style Reference (Gemini)
Download high-res versions and use as style references in your prompts:
```
"In the style of this traditional Tanjore painting [attach reference image]..."
```

### For Public Domain Images
No attribution required, but good practice to mention source.
