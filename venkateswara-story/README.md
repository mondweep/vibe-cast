# The Divine Story of Lord Venkateswara
## A Visual Storytelling Offering

This is an interactive visual storytelling experience created as a sacred offering for pilgrimage to the Tirumala Venkateswara Temple.

---

## Storytelling Plan & Structure

### Narrative Arc (13 Chapters + Opening/Closing)

| Chapter | Title | Key Content |
|---------|-------|-------------|
| **Opening** | Hero/Title | Invocation with Om symbol, date of offering |
| **1** | Vaikuntha - The Eternal Abode | Setting - Vishnu and Lakshmi in heaven |

| **3** | The Sacred Mark | The fateful kick and Lakshmi's departure |
| **4** | The Divine Separation | Vishnu's 12-year penance on Venkata Hill |
| **5** | The Divine Rebirths | Parallel stories of Padmavathi and Srinivasa |
| **6** | The Forest Encounter | The meeting of separated souls |
| **7** | The Divine Matchmaking | Narada, Vakula Devi, and the proposal |
| **8** | The Eternal Debt | Kubera's loan for the wedding |
| **9** | The Celestial Wedding | The divine marriage ceremony |
| **10** | The Temple of Seven Hills | Tirumala Temple - architecture & facts |
| **11** | The Sacred Guardians | Ramanujacharya, Krishnadevaraya, Annamacharya |

## How to Add Real Images

1. Replace placeholder divs with actual `<img>` tags
2. Use high-quality traditional art (minimum 800px width)
3. Prefer Tanjore paintings for deity images
4. Use temple photography for architectural shots
5. Maintain aspect ratios for visual consistency

Example:
```html
<!-- Replace this -->
<div class="image-placeholder" data-scene="vaikuntha">
    <div class="placeholder-icon">🏛️</div>
</div>

<!-- With this -->
<img src="images/vaikuntha-tanjore.jpg"
     alt="Vishnu reclining on Adishesha in Vaikuntha"
     class="scene-image">
```
