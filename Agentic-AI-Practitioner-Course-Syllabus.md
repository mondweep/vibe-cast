# Agentic AI Practitioner Course

## Course Overview

**Duration**: 3 months
**Sessions**: 2 hours/week (24 sessions total)
**Certification**: Students will be prepared for agentic AI practitioner roles and related certifications

## Course Philosophy

This course takes a hands-on, project-driven approach to learning agentic AI. Students will:
- Build real workflows from day one
- Use industry-standard tools (OpenCode, SimpleAgents)
- Integrate with external APIs and databases
- Deploy production-ready agentic systems
- Prepare for relevant certifications (Claude Code Architect, etc.)

## Course Schedule

### Certification Alignment

This course is intentionally mapped to the Claude Certified Architect — Foundations exam domains. The course structure is designed to develop the practical skills and architectural judgment required for this certification.

- Domain 1: Agentic Architecture & Orchestration — covered by Sessions 4, 7, 10, 15-18, 21, and 23-24
- Domain 2: Tool Design & MCP Integration — covered by Sessions 3, 6, 11, 14, 17, 19, and 20
- Domain 3: Claude Code Configuration & Workflows — covered by Sessions 2, 5-6, 9, 13, 19, and 22
- Domain 4: Prompt Engineering & Structured Output — covered by Sessions 2, 3, 5, 15, 16, 18, and 24
- Domain 5: Context Management & Reliability — covered by Sessions 4, 9, 10, 14, 18-22

### Scenario-Based Project Alignment

This course also uses certification-style scenarios as project anchors:
- Customer support resolution agent (Session 18)
- Code generation and dev workflow with Claude Code (Sessions 5-6, 15)
- Multi-agent research system (Session 17)
- Developer productivity assistant (Session 16)
- CI/CD integration with Claude Code (Sessions 13, 19)
- Structured data extraction and document processing (Sessions 15, 22)

## Sessions 5-6: OpenCode & SimpleAgents (Detailed Syllabus)

### Session 5: OpenCode — AI Coding Agent Mastery

**Duration**: 2 hours
**Prerequisites**: Python basics, command line familiarity

#### Part 1: Introduction to OpenCode (45 min)

**Topics**:
- What is OpenCode? (Zen models for coding agents)
- OpenCode vs traditional IDE coding
- Use cases: prototyping, automation, workflow building
- Zen model marketplace and transparent pricing
- Setting up OpenCode Zen account

**Hands-On**: Configure Zen account and select first model

#### Part 2: OpenCode Practical Workflows (45 min)

**Topics**:
- Navigating and executing tasks in OpenCode
- Multi-step reasoning with AI agents
- Integration with external tools and APIs
- Structured output handling

**Hands-On**: Use OpenCode to generate and validate code for a simple workflow

#### Part 3: OpenCode for Agentic AI Development (30 min)

**Topics**:
- Building agentic workflows using OpenCode
- Connecting LLMs to external data sources
- Iterative refinement and debugging

**Hands-On**: Create a simple agent that fetches and summarizes data

---

### Session 6: SimpleAgents — YAML Workflow Automation

**Duration**: 2 hours
**Prerequisites**: Session 5 concepts, basic YAML understanding

#### Part 1: SimpleAgents Core Concepts (45 min)

**Topics**:
- What is SimpleAgents? (YAML-based workflow definition)
- Key abstractions: nodes, edges, node types
- Node types: `llm_call`, `custom_worker`, `switch`, `human_input`
- Self-healing JSON and schema coercion
- Installing and running SimpleAgents

**Hands-On**: Install SimpleAgents and run first workflow

#### Part 2: Building a Real-World Workflow (45 min)

**Topics**:
- Designing the workflow graph
- Connecting to external APIs (Tavily)
- Custom worker implementation (Python)
- YAML interpolation and payload passing

**Hands-On**: Build a 3-node news collector workflow

#### Part 3: Advanced Patterns & Deployment (30 min)

**Topics**:
- Conditional routing with `switch` nodes
- Human-in-the-loop patterns
- Workflow observability (Langfuse, OpenTelemetry)
- Production deployment checklist

**Hands-On**: Add human approval step to workflow

### Session 6 Lab Exercise

**Goal**: Build the Agentic AI News Workflow

```
news_collector (custom_worker) → digest_generator (llm_call) → finalizer (custom_worker)
```

**Deliverables**:

## Certification Preparation

This course includes dedicated preparation for the Claude Certified Architect — Foundations exam.

- Weekly labs and projects are structured around the exam's five content domains.
- Final presentations include scenario-based review questions framed like the certification exam.
- Students practice identifying correct architecture tradeoffs, tool design decisions, and reliability patterns.
- A final prep session reviews sample questions, common distractors, and domain-weighted study priorities.

Recommended prep activities:
- Create a one-page comparison of agentic coordinator vs subagent orchestration patterns.
- Build and document a `CLAUDE.md` workflow configuration for a code generation pipeline.
- Design a mock MCP tool interface with clear input/output contracts and error handling.
- Write prompt templates that validate structured JSON output for document extraction.
