## Guides (Step-by-Step Workflows)

**Each guide takes 1-2 hours and produces specific outputs**

1. **STEP_1_PROFILE_ANALYSIS.md** (2 hours)
   - Career highlights extraction
   - LinkedIn audit
   - GitHub portfolio review
   - Buyer persona definition
   - Initial value proposition
   - Output: Profile summary, buyer definition, value proposition draft

2. **STEP_2_POSITIONING.md** (2 hours)
   - Role-specific positioning (3-5 versions)
   - Buyer-focused headlines
   - Executive summary refinement
   - Output: 3-5 positioning docs, final executive summary

3. **STEP_3_CV_ENHANCEMENT.md** (2 hours)
   - Add public projects section
   - Include testimonials
   - Quantify all achievements
   - Create one-page version
   - Output: Full CV + one-pager

4. **STEP_4_PERSONAL_BRANDING.md** (2 hours)
   - Build professional website
   - Deploy to Netlify
   - Update LinkedIn profile
   - Align across channels
   - Output: Live website + updated profiles

5. **STEP_5_JOB_DISCOVERY.md** (1 hour/week)
   - Search for matching roles
   - Curate to 5-10 per week
   - Prepare for user approval
   - Output: Curated job list

6. **STEP_6_APPLICATIONS.md** (2 hours/batch)
   - Draft role-specific cover letter
   - Draft personalized recruiter email
   - Get user approval
   - Submit + log
   - Output: Submitted applications + tracker update

7. **STEP_7_FOLLOW_UP.md** (1 hour/week)
   - 2-week follow-up emails
   - 4-week follow-up emails
   - Track responses
   - Schedule next batch
   - Output: Follow-up log + new batch started

## Customization Guide

### By Role
**If targeting CIO:** Emphasize digital transformation, cloud strategy, IT modernization  
**If targeting CFO:** Emphasize financial impact, cost optimization, risk management  
**If targeting COO:** Emphasize operational efficiency, scale, delivery excellence

→ See `templates/POSITIONING_TEMPLATE.md` for role-specific examples

### By Salary
**£150k+:** Mid-market, large consulting  
**£170k+:** Enterprise, top consulting  
**£200k+:** FTSE 100, major finance, top-tier consulting

→ Update `tools/JOB_SEARCH_PARAMETERS.md` with your target

### By Geography
**London only:** See `references/TARGET_COMPANIES.md` (London-based)  
**UK-wide:** Expand search, adjust location flexibility  
**International:** Adapt for different market norms

## Success Measurement

### Positioning (Week 1)
- ✅ Executive summary aligns with buyer persona
- ✅ CV reads as architect + executor
- ✅ Website deployed and live
- ✅ LinkedIn profile updated

### Job Search (Weeks 2-4)
- ✅ 5-10 curated roles per week
- ✅ 80%+ role match to criteria
- ✅ Role-specific materials per application
- ✅ User approval on all applications

### Outcomes (Week 3+)
- ✅ 50%+ response rate to applications
- ✅ 20%+ interview rate
- ✅ First-round interview within 2 weeks
- ✅ Offer within 4-8 weeks
