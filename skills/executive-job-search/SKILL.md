# Executive Job Search Skill

**Framework for securing executive-level (Director/Head/VP) roles through systematic positioning, portfolio building, and personalized applications.**

---

## Overview

This skill provides a comprehensive, reproducible approach to:
- **Profile Analysis** – Understand your unique value and positioning
- **Positioning** – Develop role-specific value propositions
- **CV Enhancement** – Build credible, measurable narrative
- **Personal Branding** – Create professional website + online presence
- **Job Search** – Discover, curate, and apply to roles systematically
- **Follow-up** – Track applications and send timely reminders

**Target:** Executive-level professionals (Director, Head, VP roles)  
**Salary Focus:** £170k+ base + bonus (customizable)  
**Timeline:** 2-4 weeks to first batch applications  
**Model:** Hybrid (curated discovery + user approval, not full automation)

---

## Key Principles

1. **Positioning First** – Fix your narrative before applications
2. **Quality Over Quantity** – 5-10 curated roles per week, not 50 generic applications
3. **Role-Specific** – Tailor cover letters and recruiter emails per role
4. **Portfolio Credible** – Leverage public work (GitHub, projects, publications)
5. **Trackable** – Log submissions, schedule follow-ups, measure outcomes
6. **Personal Branding** – Website + LinkedIn + GitHub alignment
7. **User Control** – You approve every application before sending

---

## Core Workflow

### Phase 1: Self-Discovery (Days 1-3)
1. **Profile Analysis** – LinkedIn, GitHub, portfolio review
2. **Career Highlights** – Extract 5-10 quantified achievements
3. **Buyer Identification** – Define target persona (e.g., CIO, CFO)
4. **Value Proposition** – Craft headline and positioning statement

### Phase 2: Positioning (Days 4-7)
1. **CV Enhancement** – Add public projects, credentials, testimonials
2. **Role-Specific Profiles** – Create 3-5 tailored positioning documents
3. **Executive Summary** – Write buyer-focused intro
4. **LinkedIn Alignment** – Ensure profile matches positioning

### Phase 3: Personal Branding (Days 8-10)
1. **Professional Website** – Build clean, modern personal site
2. **Deploy** – Host on Netlify (free, fast)
3. **Update Profiles** – Link website across LinkedIn, GitHub, CV

### Phase 4: Job Discovery (Ongoing)
1. **Search** – Weekly discovery of 5-10 matching roles
2. **Curate** – Present curated list to user
3. **User Approval** – Get explicit approval before proceeding
4. **Research** – Understand role, company, requirements

### Phase 5: Applications (Ongoing)
1. **Draft Materials** – Role-specific cover letter + recruiter email
2. **Review** – User reviews before sending
3. **Submit** – Track submission date and method
4. **Log** – Record in application tracker

### Phase 6: Follow-Up (Weeks 2-4+)
1. **2-Week FU** – Polite check-in for non-respondents
2. **4-Week FU** – Final follow-up before moving on
3. **Response Logging** – Track all responses (rejections, interviews, etc.)
4. **Continuous Discovery** – Start new batches while following up

## Getting Started

### Quick Start (First Day)

1. **Read**: README.md (10 min)
2. **Complete**: Profile Analysis (Step 1, 1 hour)
3. **Draft**: Value Proposition (30 min)
4. **Next**: Start Positioning (Step 2)

### Full Implementation (2 weeks)

- **Week 1**: Positioning + CV + Website (7 days)
- **Week 2**: Job Discovery + Applications (5-10 roles, approved by user)
- **Ongoing**: Follow-ups + new batches (2-3 apps/week)

## Customisation

### Target Your Search

1. Edit `tools/JOB_SEARCH_PARAMETERS.md`:
   - Salary range
   - Location(s)
   - Industry focus
   - Company types (consulting, enterprise, startup)

2. Edit `templates/POSITIONING_TEMPLATE.md`:
   - Your buyer persona (CIO, CFO, COO, etc.)
   - Key differentiators
   - Revenue impact areas
   - Governance/compliance focus (if relevant)

### Scale Your Website

1. Edit `website/index.html` – Update content
2. Edit `website/styles.css` – Change colors
3. Deploy to Netlify – Ready in 3 minutes

## Pro Tips

1. **Positioning is 80% of the work** – Get it right before applying
2. **Your buyer matters** – Focus on CIO/CFO/COO, not generic "hiring manager"
3. **Quantify everything** – "£7M+ revenue" > "contributed to revenue"
4. **Combine strategy + execution** – Advisors are a dime a dozen; architects + executors are rare
5. **Public work is credibility** – GitHub, published articles, conference talks = proof
6. **Governance angle** – Responsible AI, compliance, ethics increasingly matter
7. **Follow-up is critical** – 80% of callbacks come from 2-week + 4-week follow-ups
8. **Track everything** – Application date, method, follow-up dates, responses
