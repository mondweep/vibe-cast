# Quick Start Guide - Deploy Your Website

## 🚀 3-Minute Setup

### Step 1: Prepare Files (2 minutes)
- ✅ You have: `index.html`, `styles.css`, `netlify.toml`, `README.md`
- ✅ All files are ready to deploy
- No build process needed

### Step 2: Deploy to Netlify (1 minute)

#### **Option A: Drag & Drop (Easiest)**
1. Go to https://app.netlify.com
2. Sign up or sign in
3. Drag & drop the `website/` folder onto Netlify
4. Done! Your site is live at `name-123.netlify.app`

#### **Option B: GitHub Integration (Recommended)**
1. Push `website/` folder to GitHub
2. Go to https://app.netlify.com → **New site from Git**
3. Select your repository
4. Netlify auto-deploys on every push
5. Your site updates automatically

#### **Option C: Netlify CLI**
```bash
npm install -g netlify-cli
cd website/
netlify deploy --prod
```

---

## 🌐 Add Your Custom Domain

After deployment:

1. Go to **Site settings → Domain management**
2. Click **"Add domain"**
3. Enter your domain (e.g., `mondweep.com`)
4. Follow DNS instructions (5 mins)
5. HTTPS enabled automatically ✅

## 📝 Update Your Content

Edit `index.html` to change:
- Your photo/avatar (search for "avatar-placeholder")
- Email, phone, social links
- Career highlights
- Project descriptions
- Any text content

**Tip:** Use Cmd+F (Mac) or Ctrl+F (Windows) to find and replace.

---

## 🎨 Customize Colors

Edit `styles.css` (top of file):
```css
:root {
    --primary: #1e40af;      /* Change this blue */
    --accent: #0891b2;       /* Change this cyan */
    --text-dark: #1f2937;    /* Dark gray text */
}
```

Browse color palette at [colourlovers.com](https://www.colourlovers.com) or [coolors.co](https://coolors.co).
