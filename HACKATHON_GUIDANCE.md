# TV5Monde Hackathon Guidance

This document provides instructions for using the **Agentics Hackathon CLI** (`agentics-hackathon`) to participate in the event.

## 1. Installation

The CLI is available on npm. You can run it directly using `npx` (recommended) or install it globally.

**Recommended:**
```bash
npx agentics-hackathon <command>
```

**Global Install:**
```bash
npm install -g agentics-hackathon
```

## 2. Getting Started

### Initialize Project
To start your hackathon entry, run:
```bash
npx agentics-hackathon init
```
This will guide you through an interactive setup to choose your track and template.

### Check Status
To see your current project status:
```bash
npx agentics-hackathon status
```

### Development Tools
The CLI includes helper tools for the hackathon:
```bash
npx agentics-hackathon tools
```

### MCP Server
For agentic integration (connecting your AI agent to the hackathon platform), start the MCP server:
```bash
npx agentics-hackathon mcp
```

## 3. Support & Resources

*   **Discord:** `npx agentics-hackathon discord`
*   **Website:** `npx agentics-hackathon website`
*   **Info:** `npx agentics-hackathon info`
