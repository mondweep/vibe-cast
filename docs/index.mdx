# 🧠 AI Safety & Jailbreak Research Tutorial

**Learn the fundamentals of AI safety, model architecture, and jailbreak evaluation through interactive lessons.**

This is a comprehensive tutorial covering key concepts in AI safety research, from basic building blocks to advanced evaluation methodologies. Whether you're a researcher, engineer, or curious learner, you'll gain practical understanding of how language models work and how they can be made safer.

---

## Quick Start: Choose Your Path

### 🌱 Beginner Path (Start Here)
**Foundation Layer** — Understand the basics of how language models process text.

1. **[Module 1: Tokens & Tokenization](/modules/module-01-tokens/)** (15 min)  
   Learn what tokens are and why they matter for model input.
   - Define what a token is in NLP
   - Calculate approximate token count
   - Understand context window limits

2. **Module 2: Transformer Architecture Basics** (20 min)  
   Understand the core architecture behind modern language models.
   - How transformers process text
   - Attention mechanism overview
   - Layers and embeddings

3. **Module 3: Attention Mechanism Deep Dive** (15 min)  
   Go deeper into how attention makes models understand context.
   - Query, Key, Value vectors
   - Attention weight calculation
   - Multi-head attention

**Estimated Time**: 50 minutes  
**Difficulty**: Beginner  
**No prerequisites**

---

### 🔬 Research Path (Intermediate)
**Mathematical Foundations** — Develop the mathematical tools used in model analysis.

4. **Module 5: Linear Algebra for ML — SVD Decomposition** (25 min)  
   Learn singular value decomposition and matrix mathematics.
   - Matrix multiplication and properties
   - Eigenvalues and eigenvectors
   - SVD formula and geometric intuition
   - Working example: decomposing a 3×3 matrix

5. **Module 7: Abliteration — Removing Safety via SVD** (20 min)  
   Understand how safety constraints are surgically removed from models.
   - How abliteration works mathematically
   - SVD-based refusal direction removal
   - Consequences for reasoning capability
   - Implications for AI safety

**Prerequisites**: Modules 1-3 (foundation)  
**Estimated Time**: 45 minutes  
**Difficulty**: Intermediate

---

### 🎯 Advanced Path (Evaluation & Analysis)
**Evaluation & Testing** — Learn rigorous methods for testing jailbreak effectiveness.

8. **Module 11: StrongREJECT Benchmark** (20 min)  
   Master the state-of-the-art jailbreak evaluation framework.
   - What makes a good evaluation?
   - StrongREJECT methodology
   - Willingness vs. capability scoring
   - 313-prompt benchmark

9. **Module 12: Jailbreak Evaluation Metrics** (20 min)  
   Implement actual evaluation and interpret results.
   - Scoring functions
   - False positives and false negatives
   - Comparing attack effectiveness
   - Case study analysis

10. **Module 13: Abliterated vs Aligned Models** (15 min)  
    Compare behavior of safety-removed vs safety-aligned models.
    - Design experiments to compare models
    - Quantifying capability degradation
    - Measuring refusal rates

**Prerequisites**: Modules 1, 7, 11  
**Estimated Time**: 55 minutes  
**Difficulty**: Advanced
