Evaluate systematically:
1. Optic disc (size, shape, color, cup-to-disc ratio, neuroretinal rim)
2. Retinal vasculature (arteriole/venule ratio, tortuosity, crossing changes)
3. Macula (foveal reflex, pigmentary changes, edema, drusen)
4. Retinal background (hemorrhages, exudates, cotton-wool spots)
5. Peripheral retina (tears, detachment, lattice degeneration)
6. Vitreous (opacities, hemorrhage)
7. Grading: diabetic retinopathy scale / glaucoma staging if applicable
