## Tier 1: High Relevance for Lecture Demonstrations

### Pre-Route: Proactive Traffic Monitoring (London Agentics Meetup)
- **Repo**: [london-agentics-meetup-13aug-25](https://github.com/mondweep/london-agentics-meetup-13aug-25)
- **What**: Traffic monitoring system for Kent residents built in 40 minutes by coordinated AI agents
- **Tech**: Express.js, TypeScript, mock Google Maps/TomTom APIs
- **Agents Used**: 5 specialized agents (coordinator, research, backend, frontend, test engineer)
- **Demo Materials**: DEMO_GUIDE.md (5-min walkthrough), AGENT_COLLABORATION_REPORT.md
- **Lecture Value**: Shows the speed of agentic development and how agents collaborate on a real app; 40-minute build time is a powerful metric

### MAESTRO Security Analysis (CISO London Summit)
- **Repo**: [agentic-ai-security-demo-rela8group-ciso-london-summit](https://github.com/mondweep/agentic-ai-security-demo-rela8group-ciso-london-summit)
- **What**: AI-driven security analysis of ElizaOS using the Cloud Security Alliance MAESTRO framework
- **Key Findings**: 23 vulnerabilities across 127 files, 42,000+ lines; completed in 4 hours vs 15-20 days traditional (96% faster); $5-15K vs $50-150K (90% cheaper)
- **Framework**: 7 MAESTRO security layers covering models through agent ecosystem
- **Lecture Value**: Compelling ROI story for business audience; demonstrates governance in practice; real presentation materials included

### Sentinel: AI Agentic API Testing Platform
- **Repo**: [sentinel-api-testing](https://github.com/mondweep/sentinel-api-testing)
- **What**: Automated API testing through 7 specialized AI agents across functional, security, and performance domains
- **Tech**: Python + Rust (18-21x faster), Multi-LLM support, 540+ tests
- **Agents**: Ephemeral swarm intelligence - agents created for tasks then dissolved
- **Lecture Value**: Shows agents applied to a business-critical function (API testing); multi-language approach demonstrates practical optimization

### Auto-Analyst: AI-Powered Analytics Platform
- **Repo**: [Auto-Analyst](https://github.com/mondweep/Auto-Analyst)
- **What**: AI-powered analytics for automotive dealerships - inventory pricing and market opportunity identification
- **Tech**: Next.js/React frontend, Python APIs, interactive dashboards
- **Lecture Value**: Directly relevant to data analytics students; shows AI transforming a traditional business function

### University Pitch: LBS Knowledge Graph
- **Repo**: [university-pitch](https://github.com/mondweep/university-pitch)
- **What**: Semantic knowledge graph for London Business School (london.edu) with 3,963 nodes, 3,953 edges
- **Tech**: MGraph-DB, Python, Sentence-Transformers, OpenRouter API
- **Phases**: Crawling -> Domain Modeling -> Semantic Enrichment (sentiment, topics, personas)
- **Cost**: ~$14 for complete enrichment of 3,963 nodes
- **Lecture Value**: University context makes it relatable; demonstrates AI enrichment at low cost; phased approach shows structured delivery
