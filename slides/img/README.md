# Slide Images

Place screenshots here using the naming convention below.
The build script will embed them into the correct slide.

## Naming convention

```
slide-{NUMBER}-{short-description}.png
```

Examples: `slide-05-copilot-screenshot.png`, `slide-16-preroute-demo.png`

You can add multiple images per slide by varying the description:
`slide-10-architecture-overview.png`, `slide-10-architecture-detail.png`
