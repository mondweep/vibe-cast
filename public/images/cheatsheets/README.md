# Cheatsheets

Upload module cheatsheet PNG files into the corresponding module folder.

## Folder structure

```
public/images/cheatsheets/
├── M01/    ← VPC Deep Dive cheatsheet(s)
├── M02/    ← Hybrid Connectivity cheatsheet(s)
├── M03/    ← Transit & PrivateLink cheatsheet(s)
├── M04/    ← DNS & Route 53 cheatsheet(s)
├── M05/    ← Load Balancing & CDN cheatsheet(s)
├── M06/    ← Network Security cheatsheet(s)
├── M07/    ← Monitoring & Troubleshooting cheatsheet(s)
├── M08/    ← Network Automation cheatsheet(s)
├── M09/    ← Multi-Account Architecture cheatsheet(s)
└── M10/    ← BGP & Exam Mastery cheatsheet(s)
```

## Naming convention

Name files clearly so they sort well, e.g.:
- `M01-vpc-deep-dive-cheatsheet.png`
- `M02-hybrid-connectivity-cheatsheet.png`

## Access in the app

Files placed here are served at:
`https://aws-advanced-networking.vercel.app/images/cheatsheets/M01/filename.png`

The app has a dedicated Cheatsheets section at `/cheatsheets` where
learners can view and download all uploaded cheatsheets by module.
