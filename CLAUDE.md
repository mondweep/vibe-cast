# Pi Network Explorer App

**Project Purpose:** Build a web application that enables exploration, querying, and contribution to the π (Pi) Network—a decentralized collective intelligence platform—with real-time updates powered by PubNub.

**Tech Approach:**
- Frontend: React + TypeScript
- Backend: Node.js + Netlify Functions
- Real-time: PubNub for pub/sub messaging
- Hosting: Netlify (free tier)
- Development: BHIL AI-First methodology

**Documentation Structure:**
- `docs/prd/` — Product Requirements Documents
- `docs/spec/` — Technical Specifications  
- `docs/adr/` — Architecture Decision Records
- `docs/tasks/` — Task breakdowns and sprint planning
- `docs/prompts/` — Prompt registry and versions

**BHIL Conventions:**
- Specifications are the source of truth
- All artifacts have traceability IDs (PRD-NNN, SPEC-NNN, ADR-NNN, TASK-NNN)
- Status lifecycle: draft → in-review → approved → complete
- No implementation without approved parent SPEC
- All metrics are quantified, not vague
